#VS Code Kurulumu:
VS Code resmi sitesine girerek indirme işlemini yapıyoruz. Ekstra renklendirme ve dizayn için kendi eklentilerini yüklersin.

-> VS Code'a girerek terminal bölümüne giriş yapıyoruz.
-> Terminal üzerinden klasörümüze girerek  ardından "npm install -g expo-cli" yazıyoruz ve indirme işlemini tamamlıyoruz.
Veya bu işlemi cmd üzerinden de gerçekleştirebiliriz.

->Hemen ardından "expo init learningrn" komutu ile seçenekleri göreceğiz. (learningrn ifadesi proje adı)

->"
  blank               a minimal app as clean as an empty canvas
    blank (TypeScript)  same as blank but with TypeScript configuration
    tabs (TypeScript)   several example screens and tabs using react-navigation and TypeScript
    ----- Bare workflow -----
    minimal             bare and minimal, just the essentials to get you started
"
    şeklinde gelen seçeneklerden "minimal" olanı seçiyoruz. Çünkü yapılacak projeler o kadar büyük değil ve bireysel olduğundan daha işlevsel olacaktır. 
    Ardından indirme işlemi gerçekleşiyor.

-> learningrn adında projemis hazır halde gelmektedir.
